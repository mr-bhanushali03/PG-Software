Peity Charts

How it works

Peity (sounds like deity) is a jQuery plugin that converts an element's content into a mini svg pie, donut, line or bar chart. Peity Chart

Pie Charts
1/5 226/360 0.52/1.561 1,4 226,134 0.52,1.041 1,2,3,2,2 

**HTML**
<span class="pie">1/5</span>

**JavaScript**
$(".pie").peity("pie");
Line Charts
5,3,9,6,5,9,7,3,5,2 5,3,2,-1,-3,-2,2,3,5,2 0,-3,-6,-4,-5,-4,-7,-3,-5,-2 

**HTML**
<span class="line">5,3,9,6,5,9,7,3,5,2</span>

**JavaScript**
$(".line").peity("line")
